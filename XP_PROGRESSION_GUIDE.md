# XP Progression Guide

## Rangos del Sistema de Hexágono

El sistema de hexágono tiene 4 rangos principales para cada eje:

| Rango | XP Requerido | XP Total para Alcanzar |
|-------|--------------|------------------------|
| **BEGINNER** | 0 - 2,000 XP | Inicial |
| **INTERMEDIATE** | 2,000 - 5,000 XP | 2,000 XP |
| **ADVANCED** | 5,000 - 10,000 XP | 5,000 XP |
| **ELITE** | 10,000+ XP | 10,000 XP |

## Cuánto XP Dan las Misiones

Las misiones diarias otorgan XP según el nivel de dificultad y tu objetivo:

### Por Nivel de Objetivo
- **BEGINNER missions**: ~60 XP por misión
- **INTERMEDIATE missions**: ~80 XP por misión
- **ADVANCED missions**: ~100 XP por misión
- **ELITE missions**: ~120 XP por misión

### Ejemplo de 5 Misiones Diarias (INTERMEDIATE)
```
Misión 1 (Goal): 80 XP → balanceControl
Misión 2 (Goal): 80 XP → skillTechnique
Misión 3 (Debilidad): 70 XP → jointMobility
Misión 4 (Debilidad): 60 XP → bodyTension
Misión 5 (Bonus): 100 XP → relativeStrength
---
TOTAL: ~390 XP/día distribuido en varios ejes
```

## Días Necesarios para Subir de Rango

Completando **TODAS las 5 misiones diarias**:

### De BEGINNER → INTERMEDIATE
- XP necesario: **2,000 XP**
- XP promedio/día: **~300 XP** (nivel BEGINNER)
- **~7 días** de completar todas las misiones

### De INTERMEDIATE → ADVANCED
- XP necesario: **3,000 XP** adicionales
- XP promedio/día: **~400 XP** (nivel INTERMEDIATE)
- **~8 días** de completar todas las misiones

### De ADVANCED → ELITE
- XP necesario: **5,000 XP** adicionales
- XP promedio/día: **~500 XP** (nivel ADVANCED)
- **~10 días** de completar todas las misiones

## Progresión por Eje Individual

Cada eje del hexágono progresa **independientemente**:

- **Relative Strength**: Gana XP completando ejercicios de fuerza (push-ups, pull-ups)
- **Muscular Endurance**: Gana XP con ejercicios de resistencia (planks, holds)
- **Balance Control**: Gana XP con ejercicios de equilibrio (handstands, pistol squats)
- **Joint Mobility**: Gana XP con estiramientos y movilidad
- **Body Tension**: Gana XP con ejercicios de core y hollow body
- **Skill Technique**: Gana XP practicando skills avanzados (front lever, planche)

## Consejos para Progresar Más Rápido

1. **Completa las 5 misiones diarias** - Son la forma más eficiente de ganar XP
2. **Enfócate en tus debilidades** - Las misiones de ejes débiles te ayudan a balancear el hexágono
3. **Entrena según tu objetivo** - Las 2 misiones de objetivo principal dan el mejor XP
4. **Mantén tu racha** - Completar misiones consecutivamente puede dar bonos
5. **Usa el sistema de bonos** - Completar misiones en ejes donde estás en bajo percentil da XP extra

## Valores Visuales del Hexágono

El hexágono muestra valores de **0-10** calculados desde tu XP:

| XP | Nivel | Valor Visual | Progreso |
|----|-------|--------------|----------|
| 0-500 | BEGINNER | 0-2.5 | 0-25% |
| 500-1,000 | BEGINNER | 2.5-5.0 | 25-50% |
| 1,000-1,500 | BEGINNER | 5.0-7.5 | 50-75% |
| 1,500-2,000 | BEGINNER | 7.5-10.0 | 75-100% |
| 2,000-3,500 | INTERMEDIATE | 0-5.0 | 0-50% |
| 3,500-5,000 | INTERMEDIATE | 5.0-10.0 | 50-100% |
| 5,000-7,500 | ADVANCED | 0-5.0 | 0-50% |
| 7,500-10,000 | ADVANCED | 5.0-10.0 | 50-100% |

## Ejemplo de Progresión Real

### Día 1-7 (BEGINNER, 0 → 2,000 XP)
- Completas 5 misiones/día × 7 días = 35 misiones
- Ganas ~300 XP/día
- Total: ~2,100 XP
- **¡Subes a INTERMEDIATE en todos los ejes que entrenaste!**

### Día 8-15 (INTERMEDIATE, 2,000 → 5,000 XP)
- Completas 5 misiones/día × 8 días = 40 misiones
- Ganas ~400 XP/día
- Total: ~3,200 XP adicionales
- **¡Subes a ADVANCED!**

### Día 16-25 (ADVANCED, 5,000 → 10,000 XP)
- Completas 5 misiones/día × 10 días = 50 misiones
- Ganas ~500 XP/día
- Total: ~5,000 XP adicionales
- **¡Alcanzas ELITE!**

**Total para llegar a ELITE: ~25 días de completar todas las misiones diarias**
